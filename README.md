L'objectif de ce projet est de déterminer dans quelle mesure le langage est un marqueur de l'idéologie politique. 
J'utilise à cette fin plusieurs techniques issues du cours de Machine Learning for Natural Language Processing, dispensé en troisième année de l'ENSAE.
Les données (fichiers textes et CSV de métadonnées) sont stockées dans le fichier data qui doit être inclus à l'environnement de travail.
L'ensemble des importations nécessaires à la compilation est défini au début du Notebook.
